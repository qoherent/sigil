# Sigil 1.0

Sigil 1.0 is the first stable release of the language and agent workflow.

Included contracts:

- Sigil Language 1.0.0;
- `sigil.config` schema 1.0.0;
- `@sigil/core` 1.0.0;
- `@sigil/cli` 1.0.0;
- standalone Codex Sigil skill, originally 1.0.0 and independently advanced to
  1.1.0 for implementation-component discovery and coverage review;
- language, configuration, API, migration, workflow, and compatibility
  documentation.

The v1 release line also includes the implementation of `@sigil/lsp` 1.0.0,
prepared after the original v1.0.0 release as the shared editor-neutral
language-server boundary over `@sigil/core`.

The v1 release line also includes the implemented Sigil VS Code extension 1.0.0
as the first concrete human authoring and review surface over `@sigil/lsp`.

Each contract follows semantic versioning independently. The compatibility
matrix records supported combinations.

Version 1 intentionally excludes anchors and `sigil-indexer` implementation,
editor integrations other than VS Code, plugin packaging, semantic search,
interactive CLI workflows, `.sigil` mutation, generated diagrams, and
persistent approval records. Human approval is session-scoped.

Release acceptance requires formatting, lint, type checks, core, CLI, LSP, and
VS Code extension tests, skill validation, JSR dry runs, VSIX packaging, and the
documented brownfield pilot to pass.
