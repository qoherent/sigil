# Sigil 1.0

Sigil 1.0 is the first stable release of the language and agent workflow.

Included contracts:

- Sigil Language 1.0.0;
- `sigil.config` schema 1.0.0;
- `@sigil/core` 1.0.0;
- `@sigil/cli` 1.0.0;
- standalone Codex Sigil skill 1.0.0;
- language, configuration, API, migration, workflow, and compatibility
  documentation.

Each contract follows semantic versioning independently. The compatibility
matrix records supported combinations.

Version 1 intentionally excludes anchors and `sigil-indexer` implementation, LSP
and editor integrations, plugin packaging, semantic search, interactive CLI
workflows, `.sigil` mutation, generated diagrams, and persistent approval
records. Human approval is session-scoped.

Release acceptance requires formatting, lint, type checks, core and CLI tests,
skill validation, JSR dry runs, and the documented brownfield pilot to pass.
