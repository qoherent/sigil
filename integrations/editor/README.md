# Editor Integrations

Editor integrations are the primary human authoring surface for Sigil.

Humans should be able to write and review Sigil beside code with:

- syntax highlighting;
- inline diagnostics;
- navigation to imported components;
- collected expansion previews;
- rendered component previews;
- commands that show the context an agent would receive.

Concrete editor integrations should stay thin over shared platform packages.

The VS Code integration is an implemented v1 deliverable and the first concrete
human authoring surface. Other editor integrations remain deferred.
